# Brushlib WASM

Brush engine based on WebAssembly port of MyPaint

## Build

Built with [Emscripten](https://emscripten.org/docs/getting_started/downloads.html) 3.1.26

```sh
npm run build
```

- [libmypaint](https://github.com/mypaint/libmypaint) v1.3.0
